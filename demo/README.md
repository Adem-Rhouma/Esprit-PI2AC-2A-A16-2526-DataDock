# Démonstration du Projet

Ce dossier contient les éléments de démonstration de votre projet.

## Ce que vous devez ajouter ici :

1. **Vidéo de démonstration (`demo.mp4`)** : OBLIGATOIRE pour les projets IoT/Embarqué. Elle doit montrer l'application C++ communiquant avec le matériel Arduino. Si la vidéo est trop lourde pour GitHub, mettez un lien Google Drive ou YouTube dans le `README.md` principal.
2. **Captures d'écran (`screenshots/`)** : Ajoutez des captures d'écran de l'interface de votre application Qt (ex: Dashboard, Gestion des Navires, etc.).

Exemple de structure recommandée :
```
demo/
├── demo.mp4 (ou lien vers la vidéo)
└── screenshots/
    ├── login.png
    ├── dashboard.png
    └── gestion_peche.png
```
