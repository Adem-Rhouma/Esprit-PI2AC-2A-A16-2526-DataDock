# Schéma de Câblage

Insérez ci-dessous le schéma de câblage de votre circuit IoT (Fritzing, KiCad, ou schéma annoté).

![Schéma de câblage](../docs/schema-cablage.png)

> **Instructions pour l'étudiant** : 
> 1. Générez une image de votre schéma (ex: Fritzing) et nommez-la `schema-cablage.png`.
> 2. Placez cette image dans ce dossier `docs/`.
> 3. Si vous avez le fichier source de votre schéma (ex: `.fzz`), ajoutez-le également ici.
